# Contributors

## Tech Stack

### Frontend
- **React 18** — UI component library
- **TypeScript** — static typing for JavaScript
- **Vite (SWC)** — fast build tool and dev server
- **Tailwind CSS** — utility-first CSS framework
- **shadcn/ui + Radix UI** — accessible component primitives
- **react-router-dom v6** — client-side routing
- **TanStack Query v5** — server-state management and data fetching
- **react-hook-form + zod** — form handling and schema validation
- **framer-motion** — animation library
- **recharts** — charting and data visualization
- **lucide-react** — icon library
- **date-fns** — date formatting and manipulation

### Backend
- **Python 3.11** — primary backend language
- **FastAPI** — async REST API framework
- **Uvicorn** — ASGI server
- **sse-starlette** — Server-Sent Events streaming support
- **httpx** — async HTTP client for outbound requests
- **python-multipart** — file upload handling

### AI
- **Anthropic Claude SDK** — streaming persona chat with crisis safety classifier gate

### Storage
- **Supabase Storage** — object storage for photos and voice samples

### Testing
- **Vitest + Testing Library** — frontend unit and component tests
- **pytest + pytest-asyncio + respx** — backend async tests with HTTP mocking

### Infrastructure
- **Lovable** — hosting and deployment platform
